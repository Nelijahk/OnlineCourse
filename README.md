# OnlineCourse
## 1. Back-end. 
Основні файли лежать у back_end/project/all. Там можна знайти основні функції та методи проекту, модель бази даних розроблені мовою програмування Python. 

У back_end/project/tests зберігаються розроблені Unit тести.
## 2. Front-end. 
У front_end/OnCourse/html є приклади основних сторінок. 

У front_end/OnCourse/js лежать файли, де розроблено мінімальний функціонал сторінок з використанням JavaScript. 

У front_end/OnCourse/my-app/src зберігаються файли, де додані усі веб-сторінки та створений під них функціонал за допомогою бібліотеки React.
